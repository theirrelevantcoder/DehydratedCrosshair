Dehydrated Crosshair

Dehydrated Crosshair is a lightweight external crosshair overlay and raid helper
designed for competitive FPS gameplay.

Coded by: void
Brought to you by: Dehydrated Gaming Community

FEATURES

CROSSHAIR OVERLAY

External overlay (no injection, no hooks)

Click-through (does not block mouse input)

Works in fullscreen & borderless

Adjustable size, color, opacity, and outline

Toggle crosshair on/off instantly

Dark / Light mode UI

RAID TOOLS

Raid Planner

Quick reference for common raid costs

Raid Calculator

Calculates explosives, sulfur, gunpowder, and charcoal

UI-only helpers (no gameplay interaction)

UI & PERFORMANCE

Clean, modern interface

Banner branding included

Low CPU and memory usage

Settings auto-save supported

IMPORTANT FOLDER STRUCTURE

DO NOT move files out of the folder.

All files must remain together for the app to function.

DehydratedCrosshair
│
├─ DehydratedCrosshair.exe
├─ CrosshairOverlay.exe
├─ Dehydrated_Crosshair_Banner.png
├─ overlay_settings.json
├─ _internal\

If CrosshairOverlay.exe is missing or moved, the application will not start.

HOW TO RUN

Open the DehydratedCrosshair folder

Run DehydratedCrosshair.exe

Toggle the crosshair ON

Launch your game

No installation required.

IMPORTANT NOTES

Visual overlay only

No keyboard hooks

No memory reading

No game file modification

No injection

Always follow the rules of the game you are playing.

TROUBLESHOOTING

APP DOES NOT OPEN

Ensure all files are in the same folder

Do not run files from the build directory

Run the EXE from the main DehydratedCrosshair folder

CROSSHAIR DOES NOT SHOW

Make sure the crosshair is toggled ON

Ensure overlay_settings.json exists

Try borderless or fullscreen windowed mode
